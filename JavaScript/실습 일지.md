<h3> package.json 파일 해석</h3>
```javascript
{
     "description": "",
     "script":{
            "dev" : "parcel ./index.html", 
            "build": "parcel build ./index.html" 
                },
    
    "devDependencies": {
      "parcel": "^2.8.3"
        }
    }
    
```
**"dev" : "parcel ./index.html"** : 개발서버를 열어서 테스트 할때는 dev 명령어로 스크립트를 시작하고,

**"build": "parcel build ./index.html"** : 실제 제품화해서 배포할 때는 build 명령을 사용한다.
